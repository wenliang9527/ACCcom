# 可视化分析

ACCCOM 提供实时绘图、统计仪表盘、数据对比和差异分析四类可视化工具，支持从原始数据到结构化洞察的完整分析链路。

## 实时绘图

`PlotWindow` 实时绘制串口数据波形：

- **实时更新**：基于 `PlotViewModel` 驱动数据更新
- **自定义配置**：支持自定义字段偏移量、数据类型、缩放
- **连续数据流**：适合传感器读数、电流曲线等连续数据流的可视化分析
- **Modbus 支持**：支持 Modbus 寄存器值实时绘图

## 统计仪表盘

`StatsWindow` 串口数据统计可视化：

- **实时指标**：基于 `StatsViewModel` 展示接收速率、帧率、错误率等指标
- **图形化展示**：数据性能指标的图形化呈现
- **Modbus 支持**：Modbus 事务统计图表

## 数据对比

`CompareWindow` 逐帧对比两组数据：

- **字段级对比**：选择两个条目进行字段级逐项对比
- **差异高亮**：高亮显示差异字段
- **MCP 工具**：`compare_frames`
- **Modbus 支持**：支持 Modbus 寄存器值对比

## 数据差异

`DiffWindow` 数据差异分析：

- **差异可视化**：两组数据之间的差异可视化展示
- **并排查看**：支持并排查看和差异高亮
- **Modbus 支持**：支持 Modbus 寄存器值差异分析
